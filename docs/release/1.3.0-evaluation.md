## Known Limitations (Documented)

1. **Spatial correlation** assumes flat earth (sufficient for local operations <10km)
2. **Temporal window** fixed at 5 minutes (not configurable per category)
3. **No semantic analysis** - category matching only (no NLP)
4. **Heuristic weights** - not ML-trained, manually tuned

## Recommendations for Production Use

1. **Tune spatial threshold** based on operation type:
   - Urban: 50m (high density)
   - Rural: 200m (lower density)
   
2. **Tune temporal window** based on event type:
   - Rapid events (accidents): 2 minutes
   - Slow events (protests): 10 minutes

3. **Monitor false positive rate** in first 2 weeks:
   - Target: <5% false correlations
   - Adjust thresholds if exceeded
