```markdown
## Osk 1.0.0 - Initial Stable Release

Osk is a local-first coordination system for civilian groups operating in dynamic public environments.

### Supported Environment

- **Coordinator host:** Linux laptop
- **Python:** 3.11, 3.12, 3.13
- **Member browsers:** Chromium-class only
- **Runtime:** Local-first with no cloud dependency

### Key Features

**Coordinator Operations:**
- `osk start/stop/status` - Operation lifecycle management
- `osk start --fresh` - Clean operation starts
- `osk operator login/dashboard` - Local auth and review surface
- `osk evidence export/verify/destroy` - Evidence workflows
- `osk wipe --yes` - Broadcast wipe with clean shutdown

**Member Runtime:**
- Browser-based join via QR code
- Token-to-cookie session flow
- Offline queueing and replay
- Live alerts and GPS sharing
- Wipe command handling

**Intelligence & Review:**
- Audio transcription (Whisper/fake backends)
- Vision analysis (Ollama/fake backends)
- Location clustering
- Heuristic synthesis with corroboration
- Finding triage and review

### Known Limitations

See [docs/release/1.0.0-synthesis-limits.md](../blob/main/docs/release/1.0.0-synthesis-limits.md) for synthesis quality details.

- Heuristic (keyword-based) synthesis only
- Chromium-class browsers only
- No claim of anonymity or perfect deletion
- Disconnected-device cleanup requires follow-up

### Installation

```bash
pip install osk
osk install
osk start "My Operation"
```

See [README.md](../blob/main/README.md) for full setup instructions.
