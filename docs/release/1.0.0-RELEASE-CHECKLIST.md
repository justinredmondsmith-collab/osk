## Release Steps

### 1. Tag the Release
```bash
# Ensure on main branch
git checkout main
git pull origin main

# Create signed tag
git tag -s v1.0.0 -m "Osk 1.0.0 - Local-first civilian situational awareness"

# Push tag
git push origin v1.0.0
```

### 2. Create GitHub Release
- Go to: https://github.com/justinredmondsmith-collab/osk/releases
- Click "Draft a new release"
- Choose tag: v1.0.0
- Title: "Osk 1.0.0"
- Body: Copy from docs/release/1.0.0-release-notes.md
- Attach: None (source release)
- Publish

### 3. Update Default Branch Protection
- Ensure main branch is protected
- Require PR reviews
- Require status checks (CI)

### 4. Post-Release Announcement
- Update README badge if applicable
